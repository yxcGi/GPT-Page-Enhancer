# GPT Page Enhancer

一个只作用于 `chatgpt.com` / `chat.openai.com` 的 Chrome 扩展，用来改善 ChatGPT 页面阅读宽度和公式复制体验。

## 安装

1. 打开 Chrome，进入 `chrome://extensions/`。
2. 打开右上角「开发者模式」。
3. 点击「加载已解压的扩展程序」。
4. 选择这个目录：`/Users/yxc/Desktop/code/gpt_plugin`。
5. 刷新已经打开的 ChatGPT 页面。

## 使用

- ChatGPT 页面会出现一个悬浮工具面板。
- 拖动滑条即可调整内容宽度，范围是 `45%` 到 `96%`。
- 鼠标移动到公式区域会出现复制提示，点击公式即可复制完整 LaTeX。
- 选中文字和公式一起复制时，选区中的公式也会自动转换为 LaTeX。
- 行内公式复制为 `$...$`，块级公式复制为 `$$...$$`。
- 拖动面板左侧图标可以把悬浮窗移动到屏幕任意位置。
- 点击左侧图标可以收起或展开滑条。
- 也可以点击浏览器工具栏里的扩展图标，在弹窗中调整同一个设置。

设置会通过 Chrome sync storage 保存，刷新页面后仍然生效。
